# Final Project Considerations

You are tasked with working in groups on projects using an agentic AI tool on a topic related to how advances in AI are affecting our economy and society. You can create a dashboard, simulator, analysis, timeline, etc. to demonstrate this. 

### Agentic AI
In the past few years, AI has increased programming efficiency and productivity hugely. Nowadays, many people are using AI Agents to write code for them and automate complex workflows. In the past, you might have used AI to help write code which you copy and pasted into your scripts. In this example, the human is still driving the interaction. AI Agents are basically software systems that can perform tasks autonomously by orchestrating multiple step processes. If given a broad outline (prompt) of instructions and access to the required tools, it has **agency** to decide how to accomplish that task. Human oversight may or may not be involved. 

### Vibe Coding
Another software/programming paradigm in use today is **vibe coding**. Vibe Coding is AI-assisted software development where programmers use natural language prompts to guide the AI to create, debug, and refine code. The focus is on the output and functionality of the code or software (the vibe). Rather than line-by-line manual coding as had been done in the past, the developer oversees the AI-created code and then tweaks it accordingly. There are conflicting opinions about whether this method is good or bad but it is **highly** recommended to thoroughly stress test end product before accepting it.
